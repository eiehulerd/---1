Помощь по вкладкам

  • Загрузка файла – загрузите CSV/TXT/XLSX с разделителем ; (столбцы: fio;class;informatics;physics;mathemathics;literature;music). Данные автоматически сохранятся.
  • Редактирование – просмотр, добавление, изменение (выбрать строку → обновить), удаление ученика. Все изменения сохраняются.
  • Статистика (таблица) – средняя оценка, медиана, количество и % учеников для каждой оценки (2,3,4,5) по каждому классу и предмету + общая по школе.
  • Статистика (графики) – гистограмма распределения оценок для выбранного предмета и класса.
  • О программе – информация о разработчике.
