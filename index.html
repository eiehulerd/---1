Помощь

Загрузка: Выберите файл CSV или Excel. Данные появятся в таблице.

Редактирование: Добавляйте новые записи, удаляйте ученика (все его записи).

Статистика: Для каждого предмета выводится средняя оценка, количество и процент учеников для оценок 2,3,4,5.

Графики: Столбчатая диаграмма средней оценки по предметам.

Сохранение: Кнопка "Сохранить в CSV" сохранит журнал в файл.
