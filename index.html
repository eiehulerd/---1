Что такое медиана? Медиана — это значение, которое находится в середине упорядоченного ряда чисел. Например: оценки 2,3,4,4,5 → медиана = 4.

Помощь

Загрузка — Выберите файл CSV или Excel с оценками.

Редактирование — Добавляйте новые записи, удаляйте учеников.

Статистика — Для каждого предмета выводится средняя оценка, медиана, количество и процент учеников для каждой оценки (2,3,4,5).

Графики — Визуализация средней оценки по предметам.

Сохранение — Кнопка "Сохранить в CSV" сохранит все данные в файл.
