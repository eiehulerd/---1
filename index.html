Статистика оценок

Что такое медиана? Медиана — это среднее значение в упорядоченном ряду чисел. Например, оценки: 2,3,4,4,5 → медиана = 4 (третье число по порядку).

Помощь

📁 Загрузка: Загрузите CSV файл с оценками. Разделитель - точка с запятой (;).

✏️ Редактирование: Добавляйте новых учеников, удаляйте или редактируйте выбранного.

📊 Статистика: Показывает среднюю оценку, медиану, количество и процент учеников для каждой оценки (2,3,4,5).

📈 Графики: Визуализация распределения оценок по предметам.

💾 Сохранение: Кнопка "Сохранить в CSV" сохраняет журнал в файл.
